# 🎵 Music Genre Classification using Deep Learning (CNN)

## 📌 Overview
This project implements a **Convolutional Neural Network (CNN)** model to classify music into different genres using audio signal processing and deep learning techniques.

The model extracts features from audio files and predicts the genre automatically.

---

## 🚀 Features
- Audio preprocessing using Librosa
- MFCC / Spectrogram feature extraction
- CNN model built using TensorFlow/Keras
- Model training and validation
- Accuracy and performance visualization
- Genre prediction for new audio files

---

## 🧠 Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Librosa
- Matplotlib
- Scikit-learn
- Jupyter Notebook

---

## 📂 Dataset
The dataset contains audio files categorized into multiple genres such as:
- Classical
- Jazz
- Rock
- Pop
- Hip-Hop
- Blues
- Country
- Disco
- Metal
- Reggae

---

## ⚙️ Project Workflow
1. Load and explore dataset  
2. Extract audio features (MFCCs / Spectrograms)  
3. Prepare training and testing data  
4. Build CNN architecture  
5. Train model  
6. Evaluate performance  
7. Predict genre for new audio samples  

---

## 🏗️ CNN Architecture
- Convolutional Layers
- ReLU Activation
- MaxPooling Layers
- Dropout Layers
- Fully Connected Dense Layers
